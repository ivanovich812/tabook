Последовательность:
~~~~~~~~~~~~~~~~~~~
1)Скачиваем файл db3.sqlite3 (на всякий случай) с сервера pythonanywhere.com
2)В Pycharm делаем Commit&Push на GitHub.
3)Переходим на сервер pythonanywhere.com заходим в Consoles и запускаем консоль
4)Заходим в папку проекта cd tabook (т.е команда pwd должна вернуть нам путь /home/tabook/tabook)
5)Обновляем файлы на сервере свежими файлами из Github командой:
git pull https://github.com/ivanovich812/tabook.git
будет просить ввести логин и пароль(токен вводим просто копи-пастой) (лежат на Github в проекте token!)
6)По идее должно обновится и выдать что типа 1 file changed, 1 insertion(+), 1 deletion(-)
7) RELOAD сервер!
8)Больше ничего делать не нужно. Сайт должен работать и локально и на веб.
