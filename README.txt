Допустим, добавили пару свежих мелодий на приложение (сервер), после решили подправить локально код и залить изменения на сервер.

Идем на Pythonanywhere.com >> Console
* Для этого сначала вытянем новый файл db3.sqlite3 и картинки с сервера на локал:
1) из папки проекта /home/tabook/tabook запускаем команду git status
увидим изменения.
2) добавляем изменения в коммит
git add db3.sqlite3
git add media/images
3) снова git status
4) git commit -m 'some comment'
5) git push origin master
6) проверяем изменения на GitHub.com

* Теперь можно обновить базу данных и картинки на PyCharm.
0) остановить локальный сервер
1) делаем Update и соглашаемся с Merge.
2) проверяем актуальность мелодий в локальной версии.

* теперь можно делать Commit & Push из PyCharm ()

3)Обновляем файлы на сервере свежими файлами из Github командой:
git pull https://github.com/ivanovich812/tabook.git
будет просить ввести логин и пароль(токен вводим просто копи-пастой) (лежат на Github в проекте token!)
4)По идее должно обновится и выдать что типа 1 file changed, 1 insertion(+), 1 deletion(-)
5) RELOAD сервер!
6)Больше ничего делать не нужно. Сайт должен работать и локально и на веб.
